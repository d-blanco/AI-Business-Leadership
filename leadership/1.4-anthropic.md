## Key Concepts for Leaders

#### Skills
Reusable instruction packages you upload once that teach Claude how to handle specific tasks consistently. Think of them as onboarding playbooks for your AI assistant — write the standard once, apply it forever.

#### Projects
Workspaces that keep related conversations, files, and context together. Claude maintains consistent context across all conversations within a Project, so you never have to re-explain your role, company, or objectives.

#### Iterative Refinement
AI outputs rarely arrive perfect on the first try. The best results come from reviewing Claude's output and refining your prompt based on what's missing, wrong, or off-tone. This dialogue is the skill.

#### AI Hallucinations
When AI generates confident-sounding but factually incorrect information. All AI outputs — especially numerical analysis — must be verified before use in real business decisions.

#### Data Privacy
Free AI tools may store your inputs and use them for model training. Never enter real customer data, proprietary financials, or confidential strategy into a free tool. For this lab, you will only work with fictional data.

## Part 4 — Create a Custom Sales Reporting Skill

A Skill is a reusable instruction package you upload to Claude once. After that, Claude automatically applies the Skill whenever the task matches — you never have to re-explain your formatting preferences, report structure, or quality standards.

In this section, you will create a Skill that teaches Claude your company's sales reporting standards.

> **Skills are available on all plans**, including Free. They require Code Execution to be enabled in your settings. Skills you upload are private to your account unless you are on a Team or Enterprise plan.

### Step 1: Enable Code Execution

1. Click your profile icon (top right) and open **"Settings."**
2. Go to **"Capabilities."**
3. Ensure **"Code execution and file creation"** is toggled **ON.**
4. If it was off, toggle it on and refresh the page.

### Step 2: Create Your Skill File

A Skill is a folder containing a `SKILL.md` file. You will create this file in a plain text editor (Notepad on Windows, TextEdit on Mac, or any text editor you have).

Create a new file called **`SKILL.md`** and paste the following exactly:

```
---
name: meridian-sales-report
description: Generate Meridian Software sales analysis reports and Excel spreadsheets.
  Use when analyzing Q2 sales data, comparing regional performance, building
  spreadsheets with sales metrics, or producing executive sales summaries.
---

# Meridian Software — Sales Reporting Skill

## Company Context
Meridian Software sells two B2B SaaS products:
- Apex Suite: $4,200 per unit, primarily sold in Northeast and Southeast
- Beacon Pro: $2,800 per unit, primarily sold in West and Midwest
Regions: Northeast, Southeast, West, Midwest

## Report Structure
Every sales report must include these sections in order:
1. Executive Summary (1-2 sentences, key headline metric)
2. Regional Performance Table (Region | Product | Units | Revenue | Target | Attainment%)
3. Top Performer Callout (best region/product combination)
4. Underperformance Alert (any region with attainment below 90%)
5. Trend Observation (notable pattern across the period)
6. Recommended Action (one concrete leadership action)

## Formatting Standards
- Revenue figures: use thousands notation, e.g. $596K
- Attainment percentages: one decimal place, e.g. 102.8%
- Flag underperformance in bold or with an alert marker
- Tone: Executive-level — direct, data-driven, no filler language

## Spreadsheet Standards (when creating Excel files)
- Sheet 1: Raw data table with all columns
- Sheet 2: Regional summary (Region | Total Units | Total Revenue | Avg Attainment %)
- Sheet 3: Product summary (Product | Total Units | Total Revenue | Avg Attainment %)
- Flag cells below 90% attainment in your descriptions
- Include a totals row on all summary sheets
```

### Step 3: Package and Upload Your Skill

1. Save the file as `SKILL.md` (plain text — not `.txt` or `.docx`).
2. Create a folder on your desktop called **`meridian-sales-report`**.
3. Place the `SKILL.md` file inside that folder.
4. Compress the folder into a ZIP file:
   - **Mac:** Right-click the folder > "Compress"
   - **Windows:** Right-click > "Send to" > "Compressed folder"
5. In Claude.ai, go to your profile icon > **"Customize"** > **"Skills."**
6. Click the **"+"** button, then **"Upload a Skill."**
7. Upload your ZIP file.
8. Toggle the Skill **ON** once it appears in your Skills list.

> **Tip:** Anthropic provides a built-in **skill-creator** skill that can guide you through building Skills interactively. Enable it from the Skills section and ask Claude: *"Help me create a skill for [your workflow]."* Claude will ask clarifying questions and generate a well-structured SKILL.md for you.
